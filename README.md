一款专为 macOS 打造的极致轻量级图片查看工具。它摒弃了传统看图软件的臃肿，主打“即开即用，用完即走”的高效理念，同时集成了实用的 OCR 文字提取功能，完美兼容 Apple Silicon 与 Intel 芯片。

#### ✨ 项目背景
本项目源于解决 macOS 自带【预览】应用的痛点：不支持鼠标滚轮缩放，且关闭窗口后进程仍驻留 Dock 栏。为此，我开发了这款轻量级工具，实现了滚轮缩放、ESC 退出及关闭即退出应用的流畅体验，并额外集成了实用的OCR 文字复制功能。

#### ✨ 核心特性

- **极致轻量，用完即走**：独特的应用生命周期管理，关闭图片窗口即自动退出应用，零后台驻留，彻底释放系统资源。
- **丝滑缩放交互**：原生支持鼠标滚轮及触控板手势，可随意对图片进行平滑放大与缩小，细节查看轻松自如。
- **智能文字提取 (OCR)**：内置先进的文字识别引擎，支持一键复制图片中的文字内容，无论是截图还是扫描件，都能快速转化为可编辑文本。
- **全芯片架构兼容**：采用高性能原生代码编写，完美适配 Apple Silicon (M1/M2/M3) 与 Intel 芯片，确保在任何 Mac 设备上都能极速启动、稳定运行。
